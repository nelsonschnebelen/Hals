# Drop creative here

Any image you drop in this folder (.jpg/.jpeg/.png/.webp/.avif) is
automatically picked up and rendered in the homepage Gallery on the next
build — no code changes needed.

## Workflow
1. Save the photos you like (e.g. from Instagram) to your computer.
2. On GitHub, open this folder on branch claude/eloquent-feynman-AHGAw
   -> Add file -> Upload files -> drag them in -> Commit.
3. The site rebuilds and shows them in order.

## Tips
- Prefix filenames to control order: 01-ribeye.jpg, 02-bar.jpg, ...
- Use descriptive names; they become the image alt text for SEO/a11y.
- public/hero.jpg is the hero image — replace that file to swap the hero.
